# Ejercicio de practica de Bits

Hacer un programa que guarde hasta 8 variables boleanas en un entero.

Las variables son:

* Carro   ->  on/off     1
* Casa    ->  on/off     2
* Perro   ->  on/off     4
* Gato    ->  on/off     8
* Pez     ->  on/off    16
* Moto    ->  on/off    32
* Zapato  ->  on/off    64
* Navidad ->  on/off   128


```
carro = 1
navidad = 128

si multivariable  = carro hacer
  algo

si multivariable = navidad hacer
  otra cosa

```
